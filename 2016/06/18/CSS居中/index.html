一、水平居中

1.如果元素是inline或者inline-*（比如文本或者链接），它的父级元素display值为block，让子元素水平居中的样式是在父元素上设置如下样式:

适用于inline,inline-block,inline-table,inline-flex等

2.如果元素是块级元素，你可以设置它的左右外边距为auto,（最好设置width值，否则会充满父元素的宽度，无法显示居中）

适用于块级元素的居中（不管你想要居中的元素或者它的父级元素的width值是多少）

3.如果有两个或两个以上的块级元素要在一行水平居中，可以改变他们的display属性(inline-block或者flexbox)
设置display:inline-block

设置display:flex

4.如果有多个块级元素，每个块级元素独占一行居中，则在每个块级元素上设置margin:0 auto；同样有效。

二、垂直居中（在CSS中有点麻烦）

1.元素如果是inline或者inline-*,比如text或links

  • 如果元素是单行
    可以使元素上下内边距相等实现元素垂直居中

将要居中的行内元素上下padding值设置相等

如果padding不起作用，对于不会换行的text,可以通过设置line-height等于height来使文本居中

适用于需要居中的元素设置了高度值

  • 如果元素是多行
    上下内边距相等可以让多行text垂直居中，但如果不起作用的话，如果元素是table的子元素或者设置display属性为table，子元素display:table-cell,可以使用vertical-align，table中td默认的verticle-align:middle

如果上述方法不起作用，可以用flexbox

注意：父元素必须设置高度值（px,%等）

如果上述两种都不起作用，你可以使用”ghost element”技术，在容器中放置一个full-height的伪元素，文本在其中垂直居中

添加伪元素height:100%;width:1%;

2.如果是块级元素垂直居中

  • 你知道元素的高度吗？
    引起高度变化：比如宽度变化了，会引起高度变化；文本样式变化了也会改变高度；文本数量变化也会改变高度；对于具有固定长宽比的元素，比如image，宽度变化会引起高度变化。
    如果知道高度值的话，可以像如下设置实现垂直居中

默认的box-sizing:content-box;

不知道元素的高度

要居中的块级元素高度不确定

最简单的就是使用flexbox

display:flex实现

三、水平和垂直都居中

1.元素有固定的宽高
绝对定位后设置50%的top和left，使用负的外边距等于宽度和高度的一半进行居中

width和height有确定值

2.使用flexbox设置水平垂直居中，你需要设置两个居中属性

display:flex实现水平垂直居中

3.元素的宽高未知
如果不知道元素的宽和高，可以使用transform属性实现水平垂直居中。

width和height值不确定

body {
  background: #f06d06;
  font-size: 80%;
  padding: 20px;
}
main {
  position: relative;
  background: white;
  height: 200px;
  width: 60%;
  margin: 0 auto;
  padding: 20px;
  resize: both;
  overflow: auto;
}
main div {
  position: absolute;
  background: black;
  color: white;
  width: 50%;
  transform: translate(-50%, -50%);
  top: 50%;
  left: 50%;
  padding: 20px;
  resize: both;
  overflow: auto;
}

总结

根据不同的场景选择最佳方式实现效果！GOOD LUCK!

  • markdown

Markdown入门指南

标题

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

列表

无序列表

无序列表方式：*,+,-

有序列表

有序列表方式：1.,2.等

引用

只需要在文本前加入>这种尖括号（大于号）即可

图片与链接，区别在于!

插入链接：[]()
插入图片：![](){ImgCap}{/ImgCap}

转义字符

\ 反斜杠
` 反引号
* 星号
_ 下划线
{} 大括号
[] 中括号
() 小括号
# 井号
+ 加号
- 减号
. 英文句号
! 感叹号

粗体与斜体

粗体：用两个*包含一段文本就是粗体；
斜体：用一个*包含一段文本就是斜体

代码框

```语言名，如javascript
中间放代码，代码使用tab键缩进即可
```

分割线

只需要三个*

<
